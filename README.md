# Email-Manager
Personalized email automation tool for media outreach. 

## Setup
1. **Install Dependencies:**
   `pip install pandas openpyxl python-dotenv`
2. **Environment Variables:**
   Create a `.env` file with your `EMAIL_USER` and `EMAIL_PASS` (App Password).
3. **Data:**
   Ensure your Excel file has `First Name` and `Email` columns.

## Usage
Run the script: `python3 main.py`

## Security
Sensitive data (credentials and Excel files) is excluded from version control via `.gitignore`.
